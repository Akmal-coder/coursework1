# Проект courswork_1(Курсовая 1)
 

## Описание:
Проект courswork_1 - это проект для анализа финансовых операций, отслеживания расходов по категориям, 
расчета кэшбэка и получения актуальных курсов валют и акций.


##  Функции:
Используемые функции:

get_greeting- Приветствие в зависимости от текущего времени

read_transactions_xlsx - Функция для считывания финансовых операций из Excel

load_json_data - Возвращает данные о финансовых транзакциях из JSON

get_last_four - Функция для возвращения последний четырёх символов

get_cashback - Функция для расчёта кэшбэка

filter_by_state - Фильтрует список словарей по значению ключа 'state'

get_card_infos - Возвращает информацию о картах

get_top_transactions - Выводит топ топ-5 транзакций по сумме платежа.

get_current_exchange_rate - Функция возврата текущего курса валют

get_stock - Функция возврата текущего курса акций

get_date - Возвращает дату из формата ГГГГ-ММ-ДД в ДД.ММ.ГГГГ

filter_by_date - Фильтрует список словарей в промежутке star_date и  end_date по значению Дата платежа

### Основные модули:

1. **Главная панель (`views.py`)**
   - Приветствие по времени суток
   - Информация по банковским картам
   - Топ-5 транзакций по сумме
   - Актуальные курсы валют
   - Стоимость акций в реальном времени

2. **Аналитика кэшбэка (`services.py`)**
   - Анализ выгодных категорий для кэшбэка
   - Статистика по месяцам и категориям
   - Расчет суммарного кэшбэка

3. **Отчеты по категориям (`reports.py`)**
   - Детальный анализ расходов по категориям
   - Статистика за последние 3 месяца
   - Сравнение трат по периодам

4. **Утилиты (`utils.py`)**
   - Работа с данными Excel
   - API интеграции (валюты, акции)
   - Вспомогательные функции
